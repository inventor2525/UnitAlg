# UnitAlg
A (WIP) numpy based Linear Algebra framework inspired by Unity, for python.

# Features
- Vector3
- Quaternion
- Tranfrom (aka, Matrix4x4 / Matrix3x3)
- Plane
- Ray

# Extras
1. Can add custom type converters for other frameworks
2. Can be set to any coordinate system.
